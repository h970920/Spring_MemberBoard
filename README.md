# 회원 관리 프로그램

이 프로젝트는 Java 언어를 사용하여 패키지 구조를 따르며 간단한 회원 관리 기능을 구현한 프로그램입니다.

## 주요 기능

1. **회원 등록**: 새로운 회원을 등록할 수 있습니다.
2. **회원 조회**: 등록된 회원들의 정보를 조회할 수 있습니다.
3. **회원 수정**: 회원의 정보를 수정할 수 있습니다. (이름, 이메일, 전화번호 등)
4. **회원 삭제**: 등록된 회원을 삭제할 수 있습니다.

## 프로그램 구성

프로그램은 다음과 같은 패키지로 구성되어 있습니다:

- **controller**: 사용자의 요청을 받고 적절한 서비스를 호출하여 처리합니다.
- **dto**: Data Transfer Object의 약자로, 데이터 전송을 위한 객체를 정의합니다.
- **model**: 비즈니스 로직을 처리하는 클래스를 포함합니다.
- **repository**: 데이터베이스와의 상호작용을 담당하는 클래스를 포함합니다.
- **resources**: 설정 파일 및 기타 리소스를 저장하는 폴더입니다.
  - **mybatis**: MyBatis 관련 설정 파일을 저장하는 폴더입니다.
    - **mapper**: MyBatis Mapper XML 파일을 저장하는 폴더입니다.
      - **memberMapper.xml**: 회원 관련 SQL 매핑을 정의한 MyBatis Mapper XML 파일입니다.
    - **mybatis-config.xml**: MyBatis 설정 파일입니다.
  - **log4j.xml**: Log4j 설정 파일입니다.
  - **log4jdbc.log4j2.properties**: Log4jdbc 설정 파일입니다.

## 개발 환경

- Java
- MySQL 
- Apache Tomcat 

이 프로젝트는 기본적인 CRUD(Create, Read, Update, Delete) 기능을 구현하여 회원 관리를 제공합니다. 패키지 구조를 활용하여 각 역할에 맞는 클래스를 구현하였습니다. 추가적인 보안 및 성능 개선이 필요한 경우 해당 클래스들을 수정하여 개선할 수 있습니다.
